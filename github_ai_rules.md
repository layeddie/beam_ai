# GitHub AI Usage Rules

This document outlines guidelines to help contributors and AI assistants work effectively in this repository while following GitHub's common standards.

## 1. Repository Setup for AI Assistance
- Be sure to use a `.gitignore` file to prevent unnecessary files (e.g., IDE config, binaries) from being tracked.
- Always specify minimum requirements using a `README.md`. For example, provide setup guidelines for environments (Elixir versions, etc.).
